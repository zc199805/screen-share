# Screen Share Together - 屏幕共享看电影

和朋友一起远程看电影！完全无服务器的 P2P 屏幕共享应用。

## ✨ 功能特点

- 🔒 **完全 P2P** - 无需服务器，数据直接点对点传输
- 📺 **屏幕共享** - 共享整个屏幕（包括系统音频）
- 💬 **视频通话** - 边看电影边和朋友聊天
- 🔇 **静音控制** - 支持一键静音/取消静音
- 📱 **跨平台** - 支持 PC、Android、iOS

---

## 🚀 第一步：在电脑上启动服务器

```bash
cd "d:\remote connection\screen-share-app"
python -m http.server 8080
```

启动后会显示：`Serving HTTP on :: port 8080`

### 查看电脑IP地址

按 `Win + R`，输入 `cmd`，然后输入：
```bash
ipconfig
```

找到 **IPv4 地址**，例如：`192.168.1.100`

---

## 📱 第二步：手机安装APP

### 🤖 Android 手机安装

1. **确保手机和电脑在同一个WiFi网络**

2. 打开 **Chrome 浏览器**（必须是 Chrome）

3. 在地址栏输入：`http://电脑IP:8080`
   - 例如：`http://192.168.1.100:8080`

4. 页面加载后，点击 Chrome 右上角的 **三个点 ⋮** 菜单

5. 选择 **"添加到主屏幕"** 或 **"安装应用"**

6. 在弹窗中点击 **"添加"** 或 **"安装"**

7. 回到手机桌面，会看到 **"一起看"** 图标！

8. **以后直接点击图标就能打开，像原生APP一样！**

---

### 🍎 iOS 手机安装

1. **确保手机和电脑在同一个WiFi网络**

2. 打开 **Safari 浏览器**（必须是 Safari，不能用 Chrome）

3. 在地址栏输入：`http://电脑IP:8080`
   - 例如：`http://192.168.1.100:8080`

4. 页面加载后，点击屏幕底部的 **分享按钮** 📤
   - 就是那个 **方框+向上箭头** 的图标

5. 在分享菜单中 **向上滑动**，找到 **"添加到主屏幕"**

6. 点击右上角的 **"添加"**

7. 回到手机桌面，会看到 **"一起看"** 图标！

8. **以后直接点击图标就能打开，全屏显示，像原生APP一样！**

---

## 📖 第三步：开始使用

### 共享端（在电脑上分享屏幕）

1. 电脑浏览器打开 `http://localhost:8080`
2. 点击 **"共享屏幕"**
3. 点击 **"开始共享屏幕"**，选择屏幕并允许权限
4. 复制生成的 **连接码**，通过微信发给朋友
5. 等朋友发来 **回复码**，粘贴后点击 **"连接"**

### 观看端（在手机上观看）

1. 手机打开已安装的 **"一起看"** APP
2. 点击 **"观看屏幕"**
3. 粘贴朋友发来的 **连接码**，点击 **"下一步"**
4. 复制生成的 **回复码**，发给朋友
5. 等待连接成功，开始观看！

## ⚠️ 常见问题

### 手机无法访问？
- 确保手机和电脑连接的是 **同一个WiFi**
- 检查电脑防火墙是否阻止了端口 8080
- 确认输入的IP地址正确

### 看不到"添加到主屏幕"选项？
- Android 必须使用 **Chrome** 浏览器
- iOS 必须使用 **Safari** 浏览器

### iOS 不能共享屏幕？
- 这是苹果系统限制，iOS 只能作为 **观看端**
- 需要从 PC 或 Android 设备共享屏幕
